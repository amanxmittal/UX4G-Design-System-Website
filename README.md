## 🚀 Local Development

To run the project locally with live-reloading and automatic browser opening:

1. **Open your terminal** in the project directory.
2. **Run the following command**:
   ```bash
   npm start
   ```
3. **The website will automatically open** at: [http://localhost:3000](http://localhost:3000)

*Note: This uses `browser-sync` which will automatically reload your browser whenever you save a change to any HTML, CSS, or JSX file.*

## 📂 Project Structure

- `index.html`: The main homepage with the animated hero.
- `src/`: Contains all React components (JSX) and page logic.
- `css/`: Main design tokens and structural CSS.
- `UX4G [Page Name].html`: Specific sub-pages for foundations and patterns.
- `homepage-hero-options.html`: Explorations for different hero section designs.

## 🛠 Features

- **Animated Hero Marquee**: A living background of UX4G component snippets.
- **Multilingual Support**: Typography scales for 22 Indic scripts.
- **Compliance Primitives**: Pre-built patterns for DPDP Act, Aadhaar, and SLAs.
